UI Controls: Description and Examples

1. Radiobutton

Radiobuttons allow users to select one option from a set. Only one can be selected at a time.
  • Used for mutually exclusive choices (e.g., gender selection).
  • Grouped together visually and functionally.

2. Checkbox

Checkboxes let users select one or more options independently.
  • Used for multiple selections (e.g., interests, settings).
  • Each checkbox operates independently.

3. Text input

Text inputs allow users to enter and edit text data.
  • Used for forms, search bars, comments, etc.
  • Can be single-line or multi-line (textarea).



4. Tabs

Tabs organize content into separate views, accessible by clicking tab headers.
  • Used for navigation between related sections (e.g., profile/settings).
  • Improves usability by reducing clutter.

5. Button

Buttons trigger actions when clicked (e.g., submit, save, delete).
  • Can be styled for emphasis (primary, secondary, danger).
  • Used for form submission, navigation, etc.

6. Text label

Text labels provide information or describe other UI elements.
  • Used for field names, instructions, or status messages.
  • Improves accessibility and clarity.

7. Link

Links navigate users to other pages or resources.
  • Usually underlined and colored for visibility.
  • Can open in the same or new tab.

8. Tooltip

Tooltips display additional information when hovering over an element.
  • Used for hints, explanations, or extra details.
  • Improves user experience without cluttering UI.

9. Dropdown list

Dropdown lists let users select one option from a collapsible list.
  • Used for country selection, settings, filters, etc.
  • Can support single or multiple selections.

10. Data grid

Data grids display tabular data with sorting, filtering, and editing features.
  • Used for spreadsheets, admin panels, reports.
  • Supports pagination, search, and inline editing.
